DINOTOPIA

This website's main goal is to provide educational content on Dinosaurs through the use of beautifully illustrated images by James Gurney to immerse users in an interactive environment.

This website was created for the DECO1400 project. All images were created by James Gurney and are from his Dinotopia series. The first book, A Land Apart from Time, where the images were primarily taken from can be found here.

This website was created using some plugins, these are

jQuery Team. (n.d.). jQuery. Retrieved from https://jquery.com/
Lynch, D. (n.d.-b). maphilight. Retrieved from https://davidlynch.org/projects/maphilight/docs/
Stow, M. (n.d.). jQuery-rwdImageMaps. Retrieved from https://github.com/stowball/jQuery-rwdImageMaps

The font used was Aquifer, by JLH Fonts.

Site issues

Cookies do not work in Chrome, due to cookies not being stored in file domains. This can be fixed by either using Firefox, Edge, Safari or opening the file in the VSCode Live server extension , which will then allow cookies to work in Chrome.
